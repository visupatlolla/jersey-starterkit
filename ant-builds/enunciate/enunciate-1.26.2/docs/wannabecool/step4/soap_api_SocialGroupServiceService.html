SocialGroupServiceService

Namespace: http://api.ifyouwannabecool.com/
WSDL: api.wsdl
Path: /soap/SocialGroupServiceService

The linkage service is used to service the data for creating links between personas. This includes links and social groups.

The following methods are available on this endpoint:

  • readGroup
  • createSocialGroup
  • addToSocialGroup

readGroup

Read the specified social group.

Input Parameters

name type description
groupId string The id of the group.

Return Value

Type: socialGroup

The social group.

createSocialGroup

Create a social group.

Input Parameters

name type description
groupLeader string The id of the group leader.
memberIds string The ids of the members of the group.
exclusive boolean Whether the group is exclusive.

Return Value

Type: socialGroup

The group that was created.

addToSocialGroup

Adds a persona to a social group.

Input Parameters

name type description
groupId string The id of the group to add the persona to.
personaId string The id of the persona to add to the group.

Return Value

Type: boolean

Whether the persona was successfully added.

Faults

name description
ExclusiveGroupException If the group is exclusive.
